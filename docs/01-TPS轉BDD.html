1.1 什麼是 TPS？

TPS（Test Procedure Specification） 在本專案中以 Gherkin 格式撰寫，存放於：

Automation_testcase/Test_cases/TPS.md

每個 TC 以 Scenario: TCxx - short description 定義 Given / When / Then 步驟，使用英文（與 Inspection_App.feature、Inspection_AppSteps.cs 一致）。

1.2 三層文件鏈（必須同步）

新增或修改測試案例時，需依序維護以下四層文件：

順序文件用途
1Automation_testcase/Test_cases/TEST_PLAN.md測試目的、人工步驟、預期結果、Defect#
2Automation_testcase/Test_cases/SemiInspection_10_TestCases.md案例總表 + UI AutomationId 對照
3Automation_testcase/Test_cases/TPS.mdGherkin 程序規格（Given/When/Then）
—Automation_testcase/TPS_BDD_skill.mdAgent 轉換／維護 BDD 的 skill 指引
4Feature + Steps + PageObjects可執行的 BDD 自動化程式碼（見 Inspection_App.feature）
規則：一個 Scenario = 一個 TC；標題格式 TCxx - short description（英文）；Gherkin 步驟使用英文；標籤使用 @Functional / @Negative + 領域標籤（如 @Import、@RawData）。

完整 執行流程（7 步驟） 請見 操作手冊首頁。

1.3 TPS → Feature 範例（TC01）

TPS.md 中的 Scenario

@Functional @Import Scenario: TC01 - Import Recipe to Recipe_data Given test data is ready And the application has relaunched When I click toolbar "Import Recipe" And I select sample InspectionRecipe_Sample.json in the file dialog And I click toolbar "RawData" Then Recipe_data should contain InspectionRecipe_Sample.json And the data table should be visible And the RawData view should show filename InspectionRecipe_Sample.json And the RawData parameter table should contain "Layer1_AOI_Recipe_v1" And the RawData parameter table should contain "W-20260605-001" And the RawData parameter table should contain "Brightfield" And the log should contain "Import Recipe"

對應 Feature 檔

Automation_testcase/Project_FlaUIBDD/Testcase_Inspection_App_FlaUI_BDD/Features/Inspection_App.feature

TPS 中的每個 Scenario 直接複製到 Feature 檔，步驟文字必須與 StepDefinitions 的 regex 綁定完全一致。

1.4 StepDefinitions 與 PageObject 分工

層級檔案職責
StepDefinitionsInspection_AppSteps.cs[Given] / [When] / [Then] 綁定；呼叫 PageObject；NUnit Assert
PageObjectMainWindowPage.cs 等FlaUI 找控制項、點擊、輸入、讀取日誌
HooksTestHooks.cs啟動/關閉 App、報告初始化、截圖
HelpersConfigHelper.cs 等路徑解析、截圖、報告產生

Step 綁定範例

[When(@"I click toolbar ""(.*)""")] public void WhenIClickToolbar(string buttonName) { Main.ClickToolbar(buttonName); } [Then(@"the log should contain ""(.*)""")] public void ThenTheLogShouldContain(string keyword) { ClassicAssert.IsTrue(Workspace.LogContains(keyword)); }
