快速入口：雙擊專案根目錄的 開啟操作手冊.bat 可隨時開啟本頁。

執行用批次檔（.bat）

以下批次檔位於專案根目錄 SQA_Inspetion_App\，雙擊即可執行：

  • 開啟操作手冊.bat 以系統瀏覽器開啟本手冊（docs/index.html）
  • run_tests.bat 建置被測 App + BDD 專案，執行全部 10 個 TC
  • 執行單一測試.bat 執行指定 TC（例如 TC01）；不帶參數會提示輸入
  • 開啟測試報告.bat 開啟最新 TestResultReport.html
  • 啟動測試平台.bat 啟動 Web 控制台 http://localhost:6690/
  • build_semi.bat 僅建置被測 SemiInspectionDesktop.exe
