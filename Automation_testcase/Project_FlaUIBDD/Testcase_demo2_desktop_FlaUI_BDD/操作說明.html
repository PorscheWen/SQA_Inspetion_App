概述

本專案為 Semi Inspection Desktop 的自動化測試，以 Gherkin（Features/Demo2Desktop.feature）描述 TC01～TC10，並以 FlaUI 操作 WinForms UI。

完整五章節操作手冊請見專案根目錄 docs/index.html，雙擊 開啟操作手冊.bat 開啟。
Testcase_demo2_desktop_FlaUI_BDD/ ├── Features/ Demo2Desktop.feature（Gherkin） ├── StepDefinitions/ Demo2DesktopSteps.cs ├── PageObjects/ MainWindowPage, WorkspacePage, FileDialogPage… ├── Helpers/ Config, Dialog, Screenshot, HtmlReport… ├── Hooks/ TestHooks.cs ├── tools/FlaUIInspector/ FlaUI Inspector（UI 偵錯） ├── reports/ junit-results.xml、HTML 報告 ├── 開啟Inspector.bat ├── 開啟操作說明.bat ← 本說明 └── App.config 被測程式與 Recipe_data 路徑

上層還有 Project_FlaUIBDD/web_dashboard/ Web 控制台（port 6690）。

前置需求

  • .NET 8 SDK（專案目標 net8.0-windows）
  • Python 3（僅 Web 控制台需要）
  • 已建置被測程式：專案根目錄 build_semi.bat
  • 測試資料目錄：Recipe_data\（含 InspectionRecipe_Sample.json）
App.config 使用相對路徑（以 SQA_Inspetion_App 根目錄解析），或由 setup_env.bat 環境變數覆寫。

快速開始

1. 建置 Semi Inspection 桌面程式

cd SQA_Inspetion_App build_semi.bat

產出：SemiInspectionDesktop\bin\Debug\SemiInspectionDesktop.exe

2. 建置並執行測試（命令列）

cd Automation_testcase\Project_FlaUIBDD\Testcase_demo2_desktop_FlaUI_BDD dotnet build -c Release dotnet test -c Release --no-build

單一案例範例：

dotnet test -c Release --filter "Name~TC01"

3. Web 控制台（建議）

雙擊 ..\web_dashboard\開啟控制台.bat，瀏覽器開啟 http://localhost:6690/，可勾選 Feature、啟動 exe、執行 dotnet test 並看結果。

4. 查看 HTML 報告

測試完成後開啟：

bin\Release\net8.0-windows\reports\TestResultReport.html bin\Release\net8.0-windows\reports\SemiInspectionTestReport.html

測試案例（TC01～TC10）

規格對照：Automation_testcase\Test_cases\SemiInspection_10_TestCases.md

編號摘要
TC01Import Recipe → Recipe_data（Ctrl+I）
TC02File Tree 顯示 Recipe_data
TC03RawData 參數表

重要控制項名稱（Inspector / FlaUI）

  • treeFiles — 檔案樹
  • dataGridParameters — RawData 參數表
  • txtToolLog — 日誌區
  • 工具列：btnImportRecipe、btnParameters、btnDefectChart 等

FlaUI Inspector

  1. 確認 tools\FlaUIInspector\FlaUIInspector.exe 存在；若無，執行 tools\install-flauinspect.ps1。
  2. 雙擊 開啟Inspector.bat。
  3. 在 Inspector 選擇行程 SemiInspectionDesktop，檢查上述控制項的 Name / AutomationId。

官方下載：FlaUI/FlaUInspect（v3 主程式為 FlaUInspect.exe，本專案已提供 FlaUIInspector.exe 別名）。

App.config 主要鍵值

鍵說明

TestDataDirectoryRecipe_data 資料夾
SampleXlsx預設樣本檔（X.xlsx）
DefaultTimeout / ElementTimeout等待逾時（毫秒）
TakeScreenshotOnFailure失敗時截圖至 Screenshots/

疑難排解

找不到 SemiInspectionDesktop.exe

執行專案根目錄 build_semi.bat；路徑由 App.config 相對路徑或環境變數自動解析。

UIA Timeout / 找不到控制項

用 Inspector 確認控制項 Name 與程式一致；必要時延長 ElementTimeout。

TC01 檔案對話框失敗

測試以鍵盤輸入路徑為主；執行前關閉其他 modal 對話框。

Web 控制台無法啟動

確認已安裝 Python，且 6690 port 未被占用。

完整規格見 SemiInspection_10_TestCases.md、 TEST_PLAN.md。
